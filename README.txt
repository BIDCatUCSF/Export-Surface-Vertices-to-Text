Name:
		exportVerts.m


Installation:

- Copy the .m file into the XTensions folder in the Imaris installation directory
- You will find the function, Export Vertices in the Image Processing menu

Output: 
		a text file containing the cartesian coordinates of the vertices that make up the 
		Imaris surface object. 

Requirements:
		Requries an Imaris surface object


Dependencies:
		Matlab 2015b or later
		Imaris 7 or 8
